# Materiais sobre pipelines de datos

Este repositorio recolle materiais en galego arredor do deseño, implementación e operación de pipelines de datos, cun enfoque progresivo que vai desde os conceptos xerais ata o uso de plataformas concretas.

## Contidos principais

- [Introdución aos pipelines de datos](./intro.md)
- [Plataformas e arquitecturas para pipelines de datos](./plataformas-end-to-end.md)
- [Bloque Airflow](./Airflow/index.md)
- [Bloque Databricks](./Databricks/0.index.md)

## Estrutura actual

O repositorio organízase arredor de catro pezas principais:

1. unha introdución xeral aos pipelines de datos e ao seu papel dentro das arquitecturas Big Data
2. un bloque sobre plataformas e enfoques arquitectónicos, con especial atención ás plataformas end-to-end
3. un bloque específico de Airflow, con capítulos sobre introdución, conceptos básicos, integración no proxecto, interface, primeiros DAGs, comunicación entre tarefas e operadores específicos
4. un bloque específico de Databricks, con capítulos sobre introdución, primeiros pasos, almacenamento, notebooks, pipelines, visualizacións e DataOps/DevOps

## Evolución prevista

Este material completarase no futuro cun bloque adicional sobre **arquitectura modular on-premise**, orientado á integración de compoñentes especializados nun contorno máis desacoplado e controlado.
